# Sudoku
This is an implementation of the solveSudoku recursive function. Input any Sudoku board (some of them are in Sudoku_board.txt) and see the result of final board.

Example input:

000678912<br>
072195348<br>
198342567<br>
859761423<br>
426853791<br>
713924856<br>
961537284<br>
287419035<br>
305286109<br>
